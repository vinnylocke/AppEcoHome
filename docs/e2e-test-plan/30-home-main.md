Covers the default `/dashboard` view (`HomeMain`, labelled "Dashboard") shipped in Phases 1ÔÇô2 of `docs/plans/new-home-dashboard.md` and merged with the old Overview surface in design-overhaul **Phase 4.2**: the **4-tab switcher** (Dashboard / Locations / Calendar / Weather ÔÇö the Overview tab is gone), the legacy `?view=dashboard` **and** `?view=overview` ÔåÆ home fallthrough (neither param is in the App.tsx view whitelist any more), the garden overview grid, persona-default quick actions, the densityÔåÆ**posture** toggle (localStorage `rhozly:home:preset` since Stage 4, with the legacy `rhozly:home:density` mirrored; the Workbench posture switches the ONE hero to its console voice, adds the Attention inbox + Garden Snapshot, and puts the full task board behind "Open board ÔåÆ" on the Calendar ÔÇö the Stage-4 note below is authoritative), the compact today's-tasks section, and (Phase 2) the `home-overview` telemetry ÔÇö sensor/valve chips on area rows plus the ranked attention row.
