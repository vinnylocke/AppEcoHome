# Guides

The **Guides** section is a searchable knowledge base of plant care articles. Access it from the **Tools** tab → **Guides**, or directly at `/guides`.

It has three tabs:

| Tab | What's here |
|-----|-------------|
| **Rhozly Guides** | Official care guides curated by the Rhozly team |
| **Community Guides** | Guides written and shared by Rhozly users |
| **App Help** | AI-powered Q&A for navigating and using Rhozly |

![The Guides library — the Rhozly Guides / Community Guides / App Help tabs, tag filter, and a grid of guide cards](/doc-images/12-guides-01-library.webp)

---

## Browsing and Searching

Both tabs share the same search and filter bar:

> 📸 Screenshot: The search and filter bar showing the search input, sort buttons, and the tag filter dropdown

**Search:** Type any keyword — results filter in real time to show matching guide titles and subtitles.

**Sort:**
- **Latest** — most recently published guides first
- **Most Starred** — guides with the most ★ favourites first

**Tag filter:** Tap the **All Tags** dropdown to filter by a specific label (e.g. "herbs", "pest control", "beginners", "tomatoes"). The dropdown has its own search field for finding tags quickly.

---

## Guide Cards

Each guide appears as a card in the grid.

> 📸 Screenshot: Three guide cards in a grid showing cover gradient, author avatar, title, subtitle, tag badges, and star/comment counts

| Element | Description |
|---------|-------------|
| **Cover** | Gradient placeholder (or image if the guide has one) |
| **Author** | Name and avatar initial |
| **Title** | Guide headline |
| **Subtitle** | One-line summary |
| **Tag badges** | Up to 3 topic labels (e.g. #herbs, #watering, #beginners) |
| **★ count** | Number of users who have starred this guide |
| **💬 count** | Number of comments |
| **"You" badge** | Shown on your own guides |

Tap a card to open the guide in **Reading view**.

## Reading a Guide

![The guide reading view showing the title, author info, and the guide's sections](/doc-images/12-guides-04-reading.webp)

The reading view shows:
- **Back button** to return to the guide list
- **Title** and subtitle
- **Author** name and date published
- **Content sections** — text, images, and step sequences rendered in a clean readable layout
- **Edit button** — shown only on your own guides

### Starring a Guide

Tap the **★ Star** button to save a guide to your favourites. Starred guides are easier to find later using the "Most Starred" sort.

### Comments

Scroll to the bottom of any guide to read and leave comments.

> 📸 Screenshot: The comments section of a guide showing existing comments with reply threads

- Tap **Leave a comment** and type your comment
- Tap **Reply** on any comment to start a thread
- Your own comments have a **Delete** option

## Community Guides — Writing a Guide

Any Rhozly user can write a Community Guide to share knowledge with other growers.

Tap the **✏️ Write a Guide** button (top right of the Community Guides tab), or use [Global Quick Add](./15-navigation-quick-add.md#global-quick-add) → **Create Guide**.

> 📸 Screenshot: The Guide Editor with the title field at top, a section builder below, and tag input at the bottom

### Guide Editor

**Header fields:**
- **Title** — the main headline (required)
- **Subtitle** — a one-line summary (optional)
- **Tags** — add comma-separated topic labels (e.g. `herbs, watering, beginner`)

**Content sections:**

Guides are built from sections. Tap **+ Add Section** to choose:

| Section type | Use it for |
|-------------|-----------|
| **Text** | Paragraphs of written content — supports basic formatting |
| **Image** | Upload a photo to illustrate a point |
| **Steps** | A numbered sequence of instructions |

Each section can be moved up/down and deleted.

> 📸 Screenshot: A Steps section in the editor showing three numbered steps with title and description fields each

**Text sections:** Type or paste your content. Basic formatting is supported (bold, italic, headings).

**Image sections:** Tap to upload an image from your device. Images are stored privately and linked to the guide.

**Step sections:** Add as many steps as needed. Each step has a **title** and a **description** field.

### Publishing

Tap **Publish** to make the guide visible to all Rhozly users in the Community Guides tab.

If you want to save your progress without publishing, the guide is saved as a **draft** automatically when you navigate away.

---

## Editing Your Own Guide

Open any of your guides (they show a **"You"** badge on the card) and tap **Edit Guide** in the reading view.

The full editor opens with all existing content pre-filled. Make your changes and tap **Save** to update the published guide.

---

## Deleting a Guide

In the reading view of your own guide, tap **Edit Guide** → **Delete Guide**. A confirmation appears before permanent deletion.

---

## Rhozly's Official Guides

The **Rhozly Guides** tab contains curated guides written by the Rhozly team. These are read-only — you cannot edit or delete them — but you can:
- Star them
- Comment on them
- Use them as reference material alongside community content

## App Help

The **App Help** tab provides an AI-powered Q&A assistant that answers questions about how to use Rhozly.

![The App Help tab — search-as-you-type Q&A for using Rhozly](/doc-images/12-guides-08-app-help.webp)

Type any question about the app (e.g. "How do I add a plant to my shed?" or "What does the Planner do?") and tap **Ask**. The AI reads the feature documentation and provides a contextual answer with references to the relevant sections.

This is separate from the **Plant Doctor** (which analyses plant health from photos) — App Help is specifically for navigating and understanding Rhozly's features.
