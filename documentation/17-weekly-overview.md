# Weekly Overview

The **Weekly Overview** is Rhozly's Sunday-morning recap of the week ahead. One page, seven days, every angle covered — tasks, weather events, sowings, harvests, pruning windows, maintenance, AI tips, pest/disease risk, and pollen forecast.

If you only check Rhozly once a week, this is the page to bookmark.

Access it from the **Tools** tab → **Weekly Overview**, from the Dashboard's Week Ahead card (tap to expand), or directly at `/weekly`.

![The Weekly Overview page — a Sunday-morning recap that regenerates each week](/doc-images/17-weekly-overview-01-overview.webp)

---

## When It Updates

A new overview is generated every **Sunday at 06:00 UTC** by Rhozly's weekly cron. The page always shows the most recent overview for your home.

**Sage+** users get a **Regenerate** button at the top of the page — tap to rebuild the overview on-demand for the same week (useful if your garden state changed significantly since Sunday).

If a notification is enabled, you'll get a push on Sunday morning when the new overview lands. Toggle that under **Profile → Notifications**.

## What's On the Page

### Tasks This Week

A breakdown of every pending task across the next 7 days, grouped by task type:

| Type | What it shows |
|------|---------------|
| **Watering** | Total count + busiest day |
| **Pruning** | Count + which plants are due |
| **Harvesting** | Count + the windows open this week (see [Tasks → Harvest Windows](./03-tasks.md#harvest-windows-wave-20)) |
| **Maintenance** | Roll-up of housekeeping tasks |
| **Planting** | Anything ready to plant out |

### Weather Events

The week's frost risks, heatwaves, heavy rain days, and strong-wind windows — each pulled from your forecast. Tap a row to see which day it lands on.

### Sowings This Week

If you keep [Nursery seed packets](./21-nursery-and-sowing.md), Rhozly flags packets whose recommended sow-by window opens this week. Tap any to jump straight into logging a sowing.

### Harvest Windows Opening

Plants whose harvest window starts in the next 7 days. Useful for spotting tomatoes that are about to come into season.

### Prune Windows Opening

Same idea for pruning — plants whose ideal pruning window opens this week.

### AI Tips (Sage+)

A short list of tailored, week-specific tips generated by Rhozly AI. These factor in your hemisphere, climate, current plants, and forecast.

### Pest & Disease Risk Lines

For each at-risk plant, a one-line note with:
- The risk (e.g. "powdery mildew")
- The level (low / elevated / high)
- A specific action you can take

### Pollen Forecast

Three-pollen forecast (grass / birch / ragweed), one row per day. Hay-fever sufferers know why this matters.

## Tips

- **Tap any row** to drill into the underlying tasks or plants — the overview is a launchpad, not a dead-end.
- **Skim, don't read.** The overview is designed to be glanced at over coffee, not studied. If a section feels noisy this week, scroll past it.
- **Pair it with the Today Focus card.** Today Focus tells you what to do *today*; Weekly Overview tells you what to expect across the week.

---

## Common Questions

**Why is my overview empty?** New homes show a blank state until the next Sunday cron runs. Sage+ users can regenerate on demand.

**Can I see last week's overview?** Not from the UI — the page always shows the most recent week. Each generation does archive to history, so contact support if you need it back.

**Does the overview cost AI credits?** The Sunday cron is free for every tier. The Sage+ Regenerate button consumes one AI credit per regeneration.
